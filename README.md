# 🎯 Objective

The objective of this project is to analyze sales data, identify business trends, discover top-performing products and regions, and create an interactive dashboard to support data-driven decision-making.

This project focuses on transforming raw sales data into meaningful business insights through data cleaning, exploratory data analysis (EDA), and dashboard visualization.

---

# 📂 Dataset Overview

The dataset contains sales transaction records, including:

* Order Details
* Customer Information
* Product Categories
* Product Names
* Regions
* Sales Values
* Order Dates

The dataset was cleaned and prepared before performing analysis and visualization.

# 📈 Exploratory Data Analysis (EDA)

The following analyses were performed:

### Monthly Revenue Trend

Analyzed monthly sales performance to identify growth patterns and seasonal trends.

### Sales by Category

Compared sales across different product categories.

### Sales by Region

Identified regions contributing the highest revenue.

### Sales by Customer Segment

Analyzed purchasing behavior across customer segments.

### Top Products Analysis

Identified the highest-performing products based on sales.

### State Analysis

Compared sales performance across different states.

### Shipping Analysis

Analyzed sales distribution based on shipping modes.

# 🔑 Key Insights

### Sales Performance

* Total sales exceeded 2.26 million.
* Sales showed an overall upward trend over time.

### Regional Analysis

* The West region generated the highest revenue.
* The South region contributed the lowest sales among all regions.

### Product Analysis

* A small number of products generated a significant share of total revenue.
* Technology products contributed the highest sales.

### Customer Analysis

* Consumer customers generated the largest portion of sales.
* Corporate and Home Office segments contributed the remaining share.
